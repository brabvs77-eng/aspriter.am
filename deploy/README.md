# Deploy aspriter.am static restore

The `site/` directory is the deploy target (generated by `scripts/build-site.sh`).

## Cloudflare Pages

1. Connect this repository in [Cloudflare Pages](https://pages.cloudflare.com/)
2. **Build command:** `bash scripts/build-site.sh`
3. **Build output directory:** `site`
4. **Environment variable (optional):** `SNAPSHOT_TIMESTAMP=20230321042548`
5. Attach custom domain `aspriter.am` in Pages → Custom domains

For local-only deploy (mirror already built):

```bash
bash scripts/build-site.sh
npx wrangler pages deploy site --project-name=aspriter-am
```

## Netlify

```toml
# netlify.toml (optional)
[build]
  command = "bash scripts/build-site.sh"
  publish = "site"
```

## Notes

- Cart, checkout, and search require a live PrestaShop backend.
- Import `data/products-prestashop.csv` when rebuilding full e-commerce.
- Product images use `home_default` size (available in Wayback archive).
